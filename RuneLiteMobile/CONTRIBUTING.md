# Contributing to RuneLiteMobile

Thanks for your interest in contributing!

## Development Setup

1. Clone the repo
2. Work on the `plugin/` (Java/Gradle) and `android/` (Kotlin) folders
3. Use the existing structure and follow the patterns in the codebase

## Commit Guidelines

- Use clear, descriptive commit messages
- Keep changes focused (one feature/fix per commit when possible)
- Update documentation when making significant changes

## Pull Requests

- Open a PR with a clear description of the changes
- Reference any related issues
- Be open to feedback and iteration

## Code Style

- Follow existing code style and formatting
- Add meaningful comments for complex logic
- Keep public APIs well documented

## Questions?

Feel free to open an issue to discuss ideas before implementing.
