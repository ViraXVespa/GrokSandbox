## 1. High-Level Overview

ChatBet enables streamers to run interactive betting/polls in OSRS via in-game chat commands (!chatbet, !bet, etc.). Viewers wager virtual tokens on outcomes. The system supports parimutuel and fixed odds betting with modular activity tracking.
