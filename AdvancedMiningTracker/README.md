# Advanced Mining Tracker

Tracks rocks you mine across OSRS worlds, shows per-rock regeneration timers grouped by world, and lets you one-click hop back to a world once ore has respawned.

## Features

- Detects when **you** deplete a mining rock
- Shows a rock icon + circular countdown until respawn
- Groups timers by world (flag + world number)
- Parent world row becomes clickable when one or more rocks on that world are ready
- Clicking a ready world quick-hops via the in-game world switcher

## Development

Java 11. Open this folder in IntelliJ and run the Gradle `run` task, or:

```bash
./gradlew run
```

If you use a Jagex account, follow [Using Jagex Accounts](https://github.com/runelite/runelite/wiki/Using-Jagex-Accounts).
